About SocialChat

A modern social messaging platform developed by Mohammed Maaz A

Our Story

SocialChat was created in 2025 by Mohammed Maaz A as a personal project to build a modern, user-friendly social messaging platform. Developed entirely by a single developer, SocialChat aims to provide a seamless experience for connecting with friends, sharing moments, and engaging in meaningful conversations.

The platform combines the best features of social media and messaging apps, with a focus on privacy, performance, and user experience. Despite being developed by a single person, SocialChat strives to deliver a robust and feature-rich platform that can compete with larger social networks.

Features

Real-time Messaging

Connect with friends through instant messaging with real-time updates and notifications.

Social Feed

Share posts, photos, and updates with your network and engage with their content.

Stories

Share temporary updates that disappear after 24 hours, perfect for quick moments.

Group Chats

Create private groups to chat with multiple friends at once (coming soon).

Technology

SocialChat is built using modern web technologies to ensure a fast, responsive, and reliable experience:

  • React for the frontend user interface
  • TypeScript for type-safe code
  • Supabase for backend database and authentication
  • Tailwind CSS and shadcn/ui for styling
  • Real-time updates using WebSockets
  • Progressive Web App (PWA) capabilities for mobile-like experience
